# NestJS Learning Journey 🚀

This repository documents my complete learning journey of **NestJS** during my internship. The goal of this project was to understand how to build scalable and production-ready backend applications using the NestJS framework.

## 📚 What I Learned

During this internship, I learned the following concepts and implemented them practically:

### ✅ NestJS Fundamentals

* NestJS project structure
* Modules
* Controllers
* Services
* Dependency Injection (DI)
* Decorators
* Routing

### ✅ REST APIs

Created REST APIs using:

* GET
* POST
* PUT
* PATCH
* DELETE

Handled request and response properly using NestJS controllers and services.

### ✅ TypeORM

Learned how to work with relational databases using **TypeORM**.

Topics covered:

* Entities
* Repositories
* CRUD Operations
* Relations (basic understanding)
* Column types
* Default values
* Enums
* UUID Primary Keys

### ✅ PostgreSQL with NeonDB

Integrated **NeonDB (PostgreSQL)** with NestJS.

Implemented:

* Database connection
* Entity synchronization
* User schema creation
* Database CRUD operations

### ✅ Authentication & Security

Built a complete authentication system using:

* User Registration
* User Login
* Password Hashing with **bcrypt**
* JWT (JSON Web Token)
* Protected Routes
* Custom JWT Guard

### ✅ Request Validation

Implemented request validation using:

* class-validator
* class-transformer
* ValidationPipe
* DTO (Data Transfer Objects)

### ✅ Error Handling

Learned how to handle:

* Unauthorized requests
* Invalid input validation
* User already exists
* Internal server errors

### ✅ Project Structure

Understood the recommended NestJS architecture, including:

* Modules
* Controllers
* Services
* DTOs
* Guards
* Interfaces
* Entities
